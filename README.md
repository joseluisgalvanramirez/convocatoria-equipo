# 🚀 Convocatoria - Equipo Multidisciplinario

Página web corporativa desarrollada en HTML y CSS para la empresa ficticia **Tech Multiplataforma S.A. de C.V.**

## 📌 Descripción

Este proyecto consiste en una página web estática que presenta una convocatoria laboral para un equipo multidisciplinario enfocado en el desarrollo de software multiplataforma (web, móvil y escritorio).

El diseño incluye:
- Fondo profesional con degradado
- Secciones organizadas
- Botón de envío de CV
- Diseño limpio y responsive

---

## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (estructura preparada)
